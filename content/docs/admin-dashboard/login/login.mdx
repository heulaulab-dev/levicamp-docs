# Login Admin

Halaman login adalah pintu masuk untuk admin agar dapat mengakses dashboard pengelolaan LeviCamp. Hanya admin yang memiliki akun terverifikasi yang dapat mengakses sistem.

![Login Page](/admin-dashboard/login.jpg)

## Cara Login

<Steps>

<Step>
### Akses Halaman Login
- Buka URL admin dashboard LeviCamp
- Jika belum login, sistem akan otomatis mengarahkan ke halaman login
</Step>

<Step>
### Masukkan Kredensial
- **Username**: Masukkan username admin yang sudah terdaftar
- **Password**: Masukkan password yang benar
- Pastikan data yang dimasukkan sudah benar
</Step>

<Step>
### Verifikasi dan Masuk
- Klik tombol "Login" untuk masuk ke dashboard
- Sistem akan memverifikasi kredensial Anda
- Jika berhasil, Anda akan diarahkan ke halaman Overview
</Step>

</Steps>

## Troubleshooting

### Lupa Password
- Hubungi super admin untuk reset password
- Pastikan email yang digunakan masih aktif

### Akun Terblokir
- Akun mungkin terblokir setelah beberapa kali percobaan login yang gagal
- Hubungi super admin untuk membuka blokir akun

### Masalah Teknis
- Refresh halaman dan coba lagi
- Pastikan koneksi internet stabil
- Hapus cache browser jika diperlukan
