# Kelola Admin

Fitur Kelola Admin memungkinkan super admin untuk menambah, mengedit, dan mengelola akun admin yang memiliki akses ke dashboard LeviCamp.

![Admin Management](/admin-dashboard/admin.jpg)

## Daftar Admin

### 📋 Tampilan Utama
- **Tabel Admin**: Menampilkan semua admin terdaftar
- **Role & Permission**: Level akses setiap admin
- **Status Aktif**: Admin yang aktif atau non-aktif
- **Last Login**: Informasi login terakhir

### 👤 Informasi Admin
Setiap admin menampilkan:
- **Nama Lengkap**: Identitas admin
- **Email**: Email untuk login dan komunikasi
- **Role**: Super Admin, Admin, Staff
- **Department**: Bagian/divisi admin
- **Phone**: Nomor telepon kontak
- **Status**: Aktif, non-aktif, suspended
- **Created Date**: Tanggal akun dibuat
- **Aksi**: Edit, hapus, reset password

## Mengelola Admin

### ➕ Tambah Admin Baru
- **Data Personal**: Nama, email, telepon
- **Credential**: Username dan password
- **Role Assignment**: Tentukan level akses
- **Department**: Assign ke divisi tertentu
- **Permissions**: Set hak akses spesifik
- **Status**: Aktif atau non-aktif

### ✏️ Edit Admin
- Update informasi personal
- Ubah role dan permissions
- Reset password jika diperlukan
- Aktivasi atau deaktivasi akun
- Update department assignment

### 🔐 Keamanan Admin
- **Strong Password**: Wajib password kuat
- **Two-Factor Auth**: Optional 2FA
- **Session Timeout**: Auto logout
- **Login Monitoring**: Track aktivitas login
