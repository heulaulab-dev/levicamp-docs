# Kelola Artikel

Fitur Kelola Artikel memungkinkan admin untuk membuat, mengedit, dan mengelola artikel blog yang ditampilkan di website LeviCamp untuk memberikan informasi dan tips camping kepada customer.

![Artikel Management](/admin-dashboard/article-management.jpg)

## Daftar Artikel

### 📋 Tampilan Utama
- **Tabel Artikel**: Menampilkan semua artikel blog
- **Status Publikasi**: Published, draft, scheduled
- **Kategori**: Filter berdasarkan kategori artikel
- **Pencarian**: Cari artikel berdasarkan judul atau konten

### 📝 Informasi Artikel
Setiap artikel menampilkan:
- **Judul**: Judul artikel yang menarik
- **Author**: Penulis artikel
- **Kategori**: Tips camping, destination, gear review
- **Status**: Published, draft, scheduled
- **Views**: Jumlah pembaca
- **Created Date**: Tanggal dibuat
- **Published Date**: Tanggal publikasi
- **Aksi**: Edit, hapus, publish/unpublish

## Mengelola Artikel

### ➕ Tulis Artikel Baru
- **Judul**: Judul yang menarik dan SEO-friendly
- **Konten**: Editor WYSIWYG untuk menulis
- **Featured Image**: Gambar utama artikel
- **Gallery**: Galeri gambar pendukung
- **Excerpt**: Ringkasan artikel
- **Tags**: Tag untuk kategorisasi
- **SEO Meta**: Title dan description untuk SEO

### ✏️ Edit Artikel
- Update konten dan judul
- Ganti featured image
- Tambah atau edit gallery
- Update tags dan kategori
- Optimize SEO meta data

### 📅 Jadwal Publikasi
- **Publish Now**: Langsung publish
- **Save as Draft**: Simpan sebagai draft
